Project 2

RFID and keypad security lock system

My project focuses on creating an Arduino-based RFID and keypad security lock system. Combining the power of RFID technology and a keypad, this project aims to provide a robust and secure method of access control. The RFID module allows authorized users to swipe their RFID cards for quick and convenient access, while the keypad serves as an alternative entry method for those without RFID cards. The Arduino microcontroller acts as the brain of the system, processing input from both the RFID module and the keypad to grant or deny access accordingly. By implementing this project, I aim to enhance security measures by ensuring only authorized individuals can gain entry, making it an ideal solution for various applications like home security, office access control, and more.

Project 3

Weather Forecast

My project revolves around creating a weather forecast website using HTML, CSS, Python, and JavaScript. The goal is to develop a visually appealing and responsive interface that fetches real-time weather data from a reliable API. HTML will structure the content, CSS will handle the styling and layout, while JavaScript will enable interactivity and dynamic updates. Using Python, I will develop the backend to retrieve weather data from the API, process it, and provide the necessary information to the frontend. Users will be able to search for specific locations, view current weather conditions, and access detailed forecasts for upcoming days. By combining these technologies, I aim to deliver an informative and user-friendly weather forecast website that helps users plan their activities effectively and stay prepared for various weather conditions.
